Fibonacci php extension

DESCRIPTION
===========

This is a php7 extension providing high performance calculations of fibonacci
numbers. Made just for fun.

EXAMPLES
========

SIMPLE USAGE
-------------

    <?php
    $number = fibonacci(40);
    print $number.PHP_EOL;
    ?>
